## Agent 1 — Condition Assessor

**[Open Agent 1 →](https://mightytonylun.github.io/stewrd-heritage-copilot/agent-1/)**

Upload a photo of any part of your heritage building. The AI classifies its condition using a four-level rubric (Good / Fair / Poor / Critical) and tells you exactly what to do next.

- Works on phone — tap to take a photo straight from the camera
- Supports Anthropic Claude, OpenAI GPT-4o, Google Gemini, xAI Grok, Mistral Pixtral
- API key saved in browser — only enter it once
- Export result as a plain-text record

---

## Agent 2 — Condition Report Digitizer

**[Open Agent 2 →](https://mightytonylun.github.io/stewrd-heritage-copilot/agent-2/)**

A guided, structured form for recording condition on-site. Covers 10 sections of a standard property walk-around. Works offline once loaded — no server, no installation.

- Tablet and mobile friendly
- Session save/restore via browser storage
- Export to plain text

## What Is STEWRD

Most heritage conservation tools are built for well-resourced institutions — national museums, major landmarks. The majority of heritage custodians are not:

- Community-managed historic houses run by volunteers
- Small religious buildings maintained by congregations
- Owner-custodians of heritage-listed homes with no conservation budget
- Local government assets managed by generalists

STEWRD is a suite of five modular, free, open-source tools that together give a non-professional custodian the capability to document condition, identify urgency, and generate reports useful to funding bodies and tradespeople.

---

## No Installation

Every tool is a single HTML file. Open in any modern browser — no build step, no server, no account required.

**Mobile:** Works in Chrome and Edge on iOS and Android. Agent 1 can open your camera directly.
